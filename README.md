# Aplicando conceitos do REACTJS hooks e API externa do TMDB para manipular informações reais

### Para executar esse projeto

### Usando npm
1. Clonar o repositório
2.  ``` npm init -y ```
3.  ``` npm install ```
4.  ``` npm start ```

### Usando yarn
1. Clonar o repositório
2.  ``` yarn init -y ```
3.  ``` yarn install ```
4.  ``` yarn start ```
